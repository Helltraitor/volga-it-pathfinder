Tries to restore map using information of this and partner's graphs. Does it relative to five spots: Current node position (if they had met here) and left, right, up, and down positions relative to the current node. Could returns the empty string when it impossible to do relative to this graph (in that case you need to swap these graphs) or when algorithm cannot restore this map.

Parameters

this_startChar that represents this start on the future map
